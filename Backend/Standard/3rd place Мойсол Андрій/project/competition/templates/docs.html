post /tournament/create/
creates a tournament and returns generated schedule (createTournament)
creates a tournament and returns generated schedule

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body Tournament
Body Parameter — name(string), number_of_teams(int)

Responses

201

tournament created, json returned

400

invalid input, object invalid
Up 
get /tournament/get/{tournament_name}/{show}

returns json with schedule and each team's statistics

Path parameters

tournament_name (required)
Path Parameter — related tournament name
show (required)
Path Parameter — the filter parameter(optional), completed returns &quot;completed&quot; matches and &quot;continuing&quot; returns continuing matches

Responses

200

json returned

404

invalid input
Up

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body Score
Body Parameter — tournament_name(string), first_team_name(string), second_team_name(string), score(string)

Responses

200

score is saved

304
